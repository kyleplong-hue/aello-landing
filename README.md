# Aëllo Landing Pages

Landing pages for Aëllo Copilot - B2B sales intelligence SaaS for European automotive/machinery dealers.

## Pages

- **index.html** - Main landing page with mystery shop scoring and Copilot demo
- **aello-dealer-tour.html** - Interactive product tour showing inbox walkthrough

## Deployment

Deployed via GitHub Pages at: https://kyleplong.github.io/aello-landing/

## Features

- Multi-language support (FR, EN, NL)
- Auto-detection based on domain TLD and browser language
- Responsive design
- Animated product demos
- Mystery shop scoring visualization
- Interactive inbox walkthrough
